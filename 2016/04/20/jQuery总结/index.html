jQuery总结

jQuery是什么？

*jQuery 并不是一种新的语言，是为了简化JavaScript开发而预先封装好了对象和方法的Javascript类库，帮助开发人员建立有高难度交互的Web 2.0特性的富客户端页面，并且兼容各大浏览器。

富客户端与瘦客户端：

  • 富客户端：（Rich Internet Applications，RIA）利用具有很强交互性的富客户端技术来为用户提供一个更高和更全方位的网络体验。
    • 瘦客户端：（Thin Client）指的是在客户端-服务器网络体系中的一个基本无需应用程序的计算机终端。

jQuery的美元符号有什么作用？jQuery库中的有什么作用？jQuery库中的() 是什么?

*$() 是 jQuery库中的工厂函数，其作用：

  1. 可以通过选择器获取页面元素；

  2. 创建页面元素

  3. 将DOM对象转换为jQuery对象

$(document).ready() 是个什么函数？为什么要用它？window.onload 事件和 jQuery ready 函数有何不同？

*window.onload这样的写法作用是希望在页面加载完，自动执行定义js代码（function）。

*$(document).ready(function(){…. })这个函数是用来取代页面中的window.onload

*Onload()和$(document).ready()的区别有以下两点：

1.我们可以在页面中使用多个document.ready()，但只能使用一次onload()；

2.document.ready()函数在页面DOM元素加载完以后就会被调用，而onload()函数则要在所有的关联资源（包括图像、音频）加载完毕后才会调用。

jQuery中有哪几种类型的选择器？

从我自己的角度来讲，可以有4种类型的选择器，如下：

  1. 基本选择器(5)：有id选择器（#）,class选择器（.）,元素名选择器(element),复合选择器(“,”隔开)，通用选择器（*）;
  2. 层次选择器(4)：后代选择器（空格隔开），子代选择器（>）,下一个兄弟（+），下面所以匹配的兄弟（~）；
  3. 表单选择器（11）：匹配表单相关控件的选择器；
  4. 过滤选择器（31）：基本过滤选择器（10），内容过滤选择器（4），可见性过滤选择器（2），属性过滤选择器（7），子元素过滤选择器（4），表单对象属性过滤选择器（4）；

    $(this) 和 this 关键字在 jQuery 中有何不同？

*$(this) 返回一个 jQuery 对象，你可以对它调用多个 jQuery 方法；

*this 代表当前元素，它是 JavaScript 关键词中的一个，表示上下文中的当前 DOM 元素。你不能对它调用 jQuery 方法，直到它被 $() 函数包裹，例如 $(this)

使用 CDN 加载 jQuery 库的主要优势是什么? 当CDN上的jQuery文件不可用时，该怎么办？

  • 加载速度一般都会比本地加载速度快
  • 节省服务器的带宽，减少服务器的压力
  • CDN提供缓存服务，第一次加载之后，不需要再次加载
    如果这些CDN上的jQuery服务不可用，我们还可以通过以下代码来切换到本地服务器的jQuery版本

jQuery 中的方法链是什么？使用方法链有什么好处？

jQuery支持将多个方法链接在一起，多个animation()，这个链接也可以应用到其它方法，构成一个方法链。 使用这个方法，对于同一个元素，链接多个方法，浏览器无需多次查找HTML元素
jQuery中的.get()、[ ]和eq()方法的区别？

get()和[]两者都是将jQuery对象转换成DOM对象 get()方法是由jQuery提供的方法
*[]是利用jQuery对象是类数组对象

  • eq()方法返回的还是jQuery对象
    jQuery中bind,live,delegate与one方法的用法及区别？

bind( )方法用于将一个处理程序附加到每个匹配元素的事件上并返回jQuery对象； live( )方法将一个事件处理程序附加到与当前选择器匹配的所有元素（包含现有的或将来添加的）的指定事件上并返回jQuery对象；
one()方法将事件处理程序附加到匹配元素的指定事件并返回jQuery对象。所附加的事件处理程序最多只能执行一次； delegate()方法基于一组特定的根元素将处理程序附加到匹配选择器的所有元素（现有的或将来的）的一个或多个事件上。

jQuery与其他JS库共存问题？

*第一种方式
/*
  * 先引入其他JS库,后引入jQuery
  * * "$"符号属于其他JS库
  * 解决冲突
  * * jQuery中 - "$"符号指代jQuery
  * * jQuery中不再使用"$"符号
  */
jQuery(document).ready(function(){
console.log("this is ready.");
});
*第二种方式
jQuery(document).ready(function($){// 在当前函数中使用"$"符号 - jQuery
});// "$"符号 - 其他JS库
*第三种方式
(function($){
// "$"符号 - jQuery
})(jQuery);
// "$"符号 - 其他JS库
*第四种方式
jQuery.noConflict();
jQuery(function($){
    console.log($("p").text());
});
*第五种方式
jQuery.noConflict();
(function($){
    console.log($("p").text());
})(jQuery);
