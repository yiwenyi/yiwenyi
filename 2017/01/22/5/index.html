认真学习 switch 语法

  • javascript

今天看书的时候看到一句

switch在比较值时使用的是全等比较，不会发生类型装换

于是写了自己写了个脑残的demo

怎么想了一会儿，怎么是这个结果呢？不对啊~人家就是为了避免写太多的=号才弄出这种逻辑的呀，改！

结论：
switch后面的括号的内容要根据相应的情况改变：

  • 做相等判断的时候,要写要判断的变量,case后不用写变量；
  • 做不等判断的时候,括号后面要写true,case后要写变量。
